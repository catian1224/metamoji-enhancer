# MetaMoJi Mobile

A script to optimise [MetaMoJi ClassRoom 3](https://product.metamoji.com/classroom/) for mobile devices.

## Features

- [iOS] Disables automatic zooming when entering text

## Usage

### Bookmarklet
Works in any browser.

1. Copy the code from [MetaMoJi Mobile Bookmarklet](./dist/metamoji-mobile_bookmarklet.min.js).
2. Create a new bookmark in your browser, paste the code into the URL field, and save it.
3. Open a MetaMoJi ClassRoom 3 page and run the bookmarklet (run it each time you access the page).

### Userscript
Works on iOS Safari.

1. Install a userscript manager extension such as [Userscripts](https://apps.apple.com/jp/app/userscripts/id1463298887).
2. Enable the extension in the Settings app or Safari.
3. Download the [MetaMoJi Mobile Userscript](./dist/metamoji-mobile_userscript.user.js) into the Userscripts directory.
4. Enable the MetaMoJi Mobile Userscript.
